 ------------------------------------------ class 表达式 （选择元素） ------------------------------------------------
 1:class selector(选择器）语法选择元素：
   find_element(By.CSS_SELECTOR, CSS Selector参数) #选择单个元素
   find_elements(By.CSS_SELECTOR, CSS Selector参数) #选择多个元素
   
 2:css 选择器支持通过任何属性来选择元素，语法是用一个方括号 [] 。
      如a元素，使用 [href="http://www.miitbeian.gov.cn"] #选择 属性href值为 http://www.miitbeian.gov.cn 的元素。
      a[href^="http"]        #a标签 href属性以'http'开头
      a[href$='com']        #同上，以'com' 结束
      a[href*="miitbeian"]  #同上，包含'miitbeian'
      div[class=misc][ctype=gun]  #多个属性的限制  div标签 class='misc'  ctype='gun'
      div,span   #选择多个元素   选择div和spn 标签
   css 选择元素验证: 开发者模式下 elements  ctrl+f 在搜索框 输入验证
3：按照次序选择子节点：uth：第几个 child：小孩，子元素 
      spn:nth-child(2)    #父元素spn 的第2个子节点  或写成span:nth-of-type(2)
         nth-child(2)     #不加父节点限制，就是选择所有位置为2 的所有元素，不管什么类型
      p:nth-last-child(2) #父元素p 的倒数第2个节点  last:倒数，最后的
      p:nth-child(odd)    #父元素p 的奇数节点    odd:奇数
      p:nth-child(even)   #父元素p的偶数解    even:偶数
      h3 + span           #兄弟节点: h3后面紧跟着的兄弟节点 span
      h3 ~ span           #兄弟节点：h3后面紧跟着的所有兄弟节点 span
 -------------------------------------- frame或iframe (框架) =====================================================
 这个 iframe 元素非常的特殊， 在html语法中，frame 元素 或者iframe元素的内部 会包含一个 被嵌入的 另一份html文档。
 <frame> 是 HTML 元素，它定义了一个特定区域，另一个 HTML 文档可以在里面展示。帧应该在 <frameset> 中使用。

<frame> 的使用不应提倡，因为有一些缺点，比如性能问题，以及使用屏幕阅读器的用户缺少无障碍。比起 <frame>，<iframe> 更应该提倡。

--------------------------------  选择框 -------------------------------------------------------------
常见的选择框包括： radio（单选）框、checkbox（复选，多选）框、select（从菜单中选择）框

1:radio框选择选项，直接用WebElement的click方法，模拟用户点击就可以了。radio框及checkbox框都是input元素，只是里面的type不同而已
  #获取 用css选择器选择id 为s_radio  input标签 属性 type='radio'
  #:checked 伪类选择器 表示选择 checked 状态的元素，对 radio 和 checkbox 类型的input有效
  element = wd.find_element(By.CSS_SELECTOR, '#s_radio input[type="radio"]:checked')
  #获取选中节点的 显示数据'value'
  print('当前点选的是：' + element.get_attribute('value'))
  # 用CSS选择器选择id为s_radio 标签input 属性value='小雷老师’ 并点选小雷老师
   wd.find_element(By.CSS_SELECTOR, '#s_radio input[value="小雷老师"]').click()
2：checkbox 复选 多选：也是直接用 WebElement 的 click 方法，模拟用户点击选择。
  注意：要选中checkbox的一个选项，必须 先获取当前该复选框的状态 ，如果该选项已经勾选了，就不能再点击。否则反而会取消选择。
     # 先把已经选中的选项全部点一遍，
    elements = wd.find_elements(By.CSS_SELECTOR, '#s_checkbox input[name=teacher]:checked')
    for element in elements:
        element.click()
    # 点选小雷老师
    wd.find_element(By.CSS_SELECTOR, '#s_checkbox input[value="小雷老师"]').click()
3:select菜单选择框 （有单选和多选） 则是一个新的select标签， Selenium 专门提供了一个 Select类（方法） 进行操作。
  .select_by_value('foo')# 根据选项的value值"foo"选择元素
  .select_by_index(0)    #根据选项的 次序 （从0开始），选择元素 老版本从1开始
  .select_by_visible_text  #根据选项的 可见文本 ，选择元素
  .deselect_by_value       #根据选项的value属性值， 去除 选中元素     deselect:撤销选定，否决
  .deselect_by_index       #根据选项的次序，去除 选中元素
  .deselect_by_visible_text #根据选项的可见文本，去除 选中元素
  .deselect_all             #去除 选中所有元素
  
  Select单选框:  不管原来选的是什么，直接用Select方法选择即可
        # 导入select类
        from selenium.webdriver.support.ui import Select
        # 创建select 对象
        select = Select(wd.find_element(By.ID, 'ss_single'))
        # 通过select 对象 根据选项的可见文本‘小雷老师’选中小雷老师
        select.select_by_visible_text('小雷老师')
        
  Select多选框: 要选中某几个选项，要注意去掉原来已经选中的选项,可以用select类 的deselect_all方法，清除所有 已经选中 的选项。
              然后再通过 select_by_visible_text方法 选择 小雷老师 和 小凯老师。
        # 清除全部选项
          select.deselect_all()
